Information Retrieval of Documents (books)

 To the run the following program type the following:
     python search.py

 Note: You need to change the current path of were the directory of books exist and the path where the filtered files will be saved. Also the process might take a a few seconds (at most a minute) to start since the tables have to be created. Once you see "MMS:" then you are ready to input queries.


To stop the program simply type "//" (without the quotes)
